Speaki appears on YouTube thumbnails.

### Basic Settings

- **Language**: English, Korean, Japanese.
- **Appearance Chance**: Decide whether Speaki appears on all thumbnails or just occasionally as a surprise. (0% ~ 100%)
- **Flip Chance**: Choose the direction Speaki looks. (At 100%, Speaki looks to the left.)

### Overlay Settings

- **Position**:
  - **Smart (Default)**: Analyzes the thumbnail to find a spot that covers the content as little as possible.
  - **Random**: Speaki appears randomly on the thumbnail without thinking.
  - **Fixed Position**: Places Speaki safely in a desired corner (e.g., top-left, bottom-right) or center.
- **Size**: Set the minimum and maximum size of the appearing Speaki.
- **Tilt**: Adjust the angle Speaki tilts.
- **Opacity**: Make Speaki transparent to keep the thumbnail visible.
- **Thumbnail Color Sync**:
  - Adjusts Speaki's tone to match the thumbnail's tone.
  - **Lighting / Tint Strength**: Fine-tune how much Speaki blends in with the thumbnail color.

## Installation

---

### For Chrome

1.  **Download**: Download the latest `YouTube-Speakify-vX.X.X.zip` from the [Releases](https://github.com/soonbuger/Speakify-YouTube/releases) page of this repository and extract it.
2.  **Manage Extensions**: Go to `chrome://extensions` in your Chrome browser address bar.
3.  **Developer Mode**: Turn on the **Developer mode** switch in the top right corner.
4.  **Load Unpacked**: Click the **Load unpacked** button in the top left corner.
5.  **Select**: Select the folder you extracted in step 1 (`dist` or `YouTube-Speakify` folder).
6.  **Done!**: Visit YouTube and Speaki will welcome you! ðŸŽ‰

### For Microsoft Edge

1.  **Download**: Download the latest `YouTube-Speakify-vX.X.X.zip` from the [Releases](https://github.com/soonbuger/Speakify-YouTube/releases) page of this repository and extract it.
2.  **Manage Extensions**: Go to `edge://extensions` in your Edge browser address bar.
3.  **Developer Mode**: Turn on the **Developer mode** switch at the bottom of the left menu.
4.  **Load Unpacked**: Click the **Load unpacked** button at the top.
5.  **Select**: Select the folder you extracted in step 1 (`dist` or `YouTube-Speakify` folder).
6.  **Done!**: Visit YouTube and Speaki will welcome you! ðŸŽ‰

### For Firefox (Beta)

1.  **Download**: Download the latest `YouTube-Speakify-vX.X.X.zip` from the [Releases](https://github.com/soonbuger/Speakify-YouTube/releases) page of this repository and extract it.
2.  **Debugging Page**: Go to `about:debugging` in the address bar.
3.  **This Firefox**: Click [This Firefox](about:debugging#/runtime/this-firefox) in the left menu.
4.  **Load Temporary**: Click the **Load Temporary Add-on...** button.
5.  **Select**: Select the `manifest.json` file inside the folder you extracted in step 1.
6.  **Done!**: Visit YouTube and check for Speaki!

\*Firefox may need to be reloaded if you restart the browser.
